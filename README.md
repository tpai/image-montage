# Image Montage
Automatic load random image from imgur.

![](http://i.giphy.com/l2Sq6vQSaiUOAdbSE.gif)

# How To Run Demo

**Install Dependencies**

```
npm i
```

**Set Environment Variable**

```
cp .env.example .env
```

.env

```
CLIENT="CLIENT ID xxxxxxxx"
```

**Run Server**

```
npm start
```
