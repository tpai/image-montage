# Image Montage
Automatic load random image from imgur.

![](http://i.giphy.com/l0MYDm8HCJ46uQM5a.gif)

## How To Run Demo

**Install Dependencies**

```
npm i
```

**Set Environment Variable**

.env

```
CLIENT="CLIENT ID xxxxxxxx"
```

**Run Server**

```
npm start
```

## How To Develoap / Build

**Babel Watch**

```
npm run dev
```

**Minify With Babili**

```
npm run build
```
